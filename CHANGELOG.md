# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto se adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2023-10-27

### Añadido
- Estructura inicial del proyecto con Next.js 14 y FastAPI.
- Integración de Prisma con SQLite para persistencia local.
- Motor de orquestación de agentes (Técnico, Fundamental, Sentimiento).
- Dashboard Premium con Tailwind CSS y shadcn/ui.
- Gráficos de velas con Lightweight Charts.
- Motor de Paper Trading con cálculo de P&L en tiempo real.
- Suite de pruebas unitarias y de integración con Jest.
